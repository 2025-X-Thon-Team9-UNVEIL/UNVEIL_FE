

## 🔹 서비스 소개


<h3>UNVEIL은 청년의 자취 생활에서 마주하는 '보이지 않는 리스트'를 시각화하는 안전 플랫폼입니다.</h3>

1. **휴대폰 기반 방음 구조 진단 (Echo Zone)** .사용자가 발생시킨 음향을 분석하여, 벽체가 단단한 콘크리트인지 얇은 가벽인지 벽체의 밀도와 구조적 방음 취약성을 객관적인 '내구력 점수'**로 진단합니다.

2. **귀갓길 위험 구간 시각화 (Light Zone)** 거리뷰 기반 분석을 통해 CCTV 사각지대와 가로등 조도가 낮은 곳을 식별하여, 사용자의 귀갓길 중 위험이 높은 'Red Zone' 구간을 지도 위에 구체적으로 밝혀줍니다
3. **생활 안전 통합 점수 산출** 방음 구조 진단 수치와 귀갓길 위험 구간 점수를 종합하여, 안전 판단 기준을 제공합니다.
